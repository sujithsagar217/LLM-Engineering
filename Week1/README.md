# Week 1 — Scripts

This folder contains the Day 1 and Day 2 practice scripts and a scraper utility.

- Day 1 — day1.py — a practice script demonstrating basic data input/output and simple transformations
- Day 2 — day2.py — exercises in string processing and regex with small test examples
- Scraper — scraper.py — a utility to fetch and parse HTML using requests and BeautifulSoup (install dependencies if needed)

Run the scripts with Python 3: python day1.py, python day2.py, or python scraper.py.
